Setup

When this plugin was installed, it automatically created a field group called SEO. You’ll need to manually add this field group to all sections that you’d like customizable SEO for. Go to Sections

Your Site Name will be used as the <title> for the homepage, and will be added to the title for all other pages, so make sure you set that appropriately. Edit Site Name
